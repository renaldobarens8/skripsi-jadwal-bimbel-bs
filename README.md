# Skripsi Jadwal Bimbel BS

Web-based tutoring schedule management system built as undergraduate thesis project.

## Features
- Role-based authentication (Teacher / Student)
- Teachers can create, edit, and delete class schedules
- Students have read-only access to view schedules
- Interactive calendar UI with per-session notes

## Tech Stack
- Laravel (PHP)
- MySQL
- JavaScript
- Vite
- XAMPP (local development)

Under active development — expected completion May 2026.
